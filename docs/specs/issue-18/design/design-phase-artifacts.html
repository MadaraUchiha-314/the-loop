The designer iteration loop

  1. Generate / curate — produce the artifact from locked requirements (HTML prototype and/or Figma).
  2. Render, don't read — review the rendered output in a browser / Figma, not the markup.
  3. Designer review — the designer persona reviews; every opinion lands as a ticket comment (paper trail).
  4. Iterate — fold feedback back by editing the checked-in artifact, not making new copies.
  5. Lock & hand off — status: approved + screenshots as evidence; the locked artifact is the visual contract implementation matches.
